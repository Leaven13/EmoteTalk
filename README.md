# EmoteTalk

adds auto emotes when you type certain words in chat

## what it does

type "hi" and your character waves. (omg)

only works on your own messages so you won't spam emotes at other people

## how to use

just install it and type normally. the emotes trigger automatically when you use keywords

## requirements

- GDWeave
- Socks
